1) *** Iniciar uma pasta no GIT

 GIT INIT

2)***Mudar o nome do ramificação de Main para Master

GIT BRANCH -M MAIN

3) ****   Conectar um repositorio , link o repositorio do computador com o gitbub

Dica no proprio github   

**** Ver qual pasta esta trabalhando:

GIT REMOTE -V

*** Saber o estado do arquivo:

GIT STATUS

*** Modificar o estado de um arquivo:

GIT ADD 	"nome do arquivo"

***  Para commit arquivo:

GIT COMMIT -M   "MENSSAGEM  opcional para saber o que foi feito"

*** Para Subir os arquivos pra o gitHub:

GIT PUSH -U ORIGIN MAIN
